# Environment

**What belongs here:** Required env vars, external dependencies, setup notes.
**What does NOT belong here:** Service ports/commands (use `.factory/services.yaml`).

---

## Database

- **PostgreSQL 18** running in Docker container `sub2api-postgres` via OrbStack
- Host: `sub2api-postgres.orb.local` (OrbStack DNS)
- Port: 5432
- User: `sub2api`
- Database: `involute`
- Connection string: see .env file (auto-generated by init.sh)
- Password is extracted at runtime from container env by init.sh

## Environment Variables

- `DATABASE_URL` — Prisma connection string (auto-generated by init.sh)
- `AUTH_TOKEN` — Service token for GraphQL API authentication (default: `involute-dev-token-001`)
- `PORT` — API server port (default: 4200)

## Node.js Toolchain

- Node.js v24.13.0 (via nvm)
- pnpm v10.30.1
- vitest for testing

## Linear API (for import feature)

- Requires `LINEAR_API_TOKEN` env var when running export commands
- Token obtained from Linear Settings → API → Personal API keys
- Not needed for development — only for actual data import
